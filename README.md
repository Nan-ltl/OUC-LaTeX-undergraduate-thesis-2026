本模板适用于 **2026 年中国海洋大学本科毕业论文**，仅供参考使用。

⚠️ 由于作者在制作过程中可能存在理解偏差或实现误差，如有格式细节不符合学校要求，请自行根据官方规范进行修改。

\---

## 🛠️ 编译环境

* LaTeX 编译器：**XeLaTeX**
* 文献工具：**Biber**
* 编辑器：**TeXstudio 4.8.6**

---

## ☁️ Overleaf 使用说明

本模板已在 Overleaf 环境下测试：

* 使用 **2025 年版本 XeLaTeX** 可正常编译
* 但编译时间超出了免费计划的编译时限，若无VIP建议使用本地编译器编译

---

## 📂 模板特点

* 支持中文排版（基于 `ctex`）
* 自定义标题格式（含黑体编号控制）
* 内置参考文献格式配置
* 示例包含：

  * 中文/英文摘要
  * 公式、表格、图片引用
  * 一些看起来很严谨的胡说八道内容
